# Rule-Based Chatbot

## Project Description
This project is a Rule-Based Chatbot developed using Python as part of the CodSoft Artificial Intelligence Internship.

## Features
- Responds to greetings
- Answers predefined questions
- Uses if-else statements
- Exits when the user types "bye"

## Technologies Used
- Python 3.14
- Visual Studio Code

## How to Run
1. Open `chatbot.py` in VS Code.
2. Run the program using:
   ```
   python chatbot.py
   ```
3. Start chatting with the bot.

## Sample Questions
- hi
- hello
- how are you
- what is your name
- what is ai
- thank you
- bye
